eCOM

This Application serves as a dummy online shopping store. It consists of the following menus :
Customer Menu :
This menu is used to create a customer who intends to purchase a product from the application. Other functionalities include View , Edit, Delete and Make an order.
Category Menu :
This menu is used to create a Category (products offered are grouped into Categories) on the application. Other functionalities include View and Edit.
Product Menu :
This menu is used to create Products which are grouped to Categories on the application. Each product is also assigned a code and default status of "AVAILABLE" .Other functionalities include View and Delete ( Only Products with status of "AVAILABLE" can be deleted )
Order Menu :
This menu is used to view orders made by customers on the application. Each Order when created is assigned an Order Code with a status of "PENDING" then changed to "COMPLETED" after processing
Comments Menu :

Technologies Used :

The following Java Technology Stack were used for this project
Primefaces 6.0 Including JSF 2.2 and Facelets
CDI Beans : Glue between the Front-end and JAX-RS calls
JAX-RS Service : Exposed the Business service methods for the CDI Beans
EJB and JPA Handles Business functions for the application.
Hibernate OGM : Used to map the JPA Entities to MongoDB database which stores records .
This application is hosted on Red Hat Openshift platform , deployed with Wildfly Application Server 9 and MongoDB 2.4 .
